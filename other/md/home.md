# VoidBot

VoidBot, Go ile yazılmış açık kaynak bir Discord botudur ve topluluk destekli olmayı hedefler.

---

## Özellikler

- Slash ve prefix komut desteği
- YAML tabanlı tam lokalizasyon sistemi
- MySQL veritabanı + Redis cache
- Docker ile kolay kurulum _(yakında)_
- Web UI _(yakında)_

---

## Teknolojiler

|            |           |
| ---------- | --------- |
| Dil        | Go        |
| Veritabanı | MySQL     |
| Cache      | Redis     |
| Discord    | discordgo |
